    Progressive Enhancement is a kind of method that is used for developing web applications. It focus on content of web, And gradually add more detailed decorations and functions on this basis. The content is marked up in HTML, and the decorations and functions are supported by CSS and JavaScript.

    It makes up for the defects of graceful degradation and allows everyone to access the basic content and functionality of a web page.
